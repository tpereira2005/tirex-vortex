# TIREX — Pneu Inteligente VORTEX

Website institucional estático desenvolvido para apresentação académica do projeto VORTEX.

## Abrir localmente

Abra `index.html` diretamente ou inicie um servidor HTTP simples nesta pasta.

## Publicar no GitHub Pages

1. Crie um repositório público no GitHub e envie estes ficheiros para a branch `main`.
2. Em **Settings → Pages**, escolha **Deploy from a branch**.
3. Selecione `main` e a pasta `/ (root)`.

O contacto apresentado é demonstrativo e deve ser substituído por um número autorizado antes de ser ativado.
